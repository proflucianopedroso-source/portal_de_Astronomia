Insira uma data e hora — a calculadora retorna o JD (Dia Juliano), MJD (Modified JD), JDN e partes inteiras/fracionárias.

Forma pedagógica e usual (passo a passo): 1) Ajuste de mês/ano (se necessário): $$\text{Se } M\le 2:\quad Y' = Y - 1,\quad M' = M + 12.\quad \text{Caso contrário: } Y'=Y,\; M'=M.$$ 2) Calcule: $$A = \left\lfloor\dfrac{Y'}{100}\right\rfloor,\qquad B = \begin{cases}2 - A + \left\lfloor\dfrac{A}{4}\right\rfloor,&\text{se calendário gregoriano}\\[4pt] 0,&\text{se calendário juliano}\end{cases}$$ 3) Defina o dia com fração (D): $$D = \text{dia} + \dfrac{h}{24} + \dfrac{m}{1440} + \dfrac{s}{86400}.$$ 4) Algorítimo final (use Y' e M'): $$\mathrm{JD} = \left\lfloor 365.25\,(Y' + 4716) \right\rfloor + \left\lfloor 30.6001\,(M' + 1) \right\rfloor + D + B - 1524.5.$$ Observação: o Dia Juliano (JD) é contínuo e começa ao meio-dia UT; o Julian Day Number (JDN) é o número inteiro do dia: $$\mathrm{JDN} = \left\lfloor \mathrm{JD} + 0.5 \right\rfloor.$$
