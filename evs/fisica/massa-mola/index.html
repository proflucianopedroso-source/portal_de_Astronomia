Ver equações e observações

Modelo linear (pequenas oscilações):

\[ m\,\ddot{x} + c\,\dot{x} + kx = 0, \quad \omega_n = \sqrt{\tfrac{k}{m}}, \quad \zeta = \tfrac{c}{2\sqrt{mk}}. \]

Para \(\zeta<1\) (subamortecido): \[ \omega_d = \omega_n\sqrt{1-\zeta^2}, \qquad T_\text{lin} = \frac{2\pi}{\omega_d}. \]

Modelo não linear fraco (Duffing):

\[ m\,\ddot{x} + c\,\dot{x} + kx + \alpha x^3 = 0. \]

Para amplitudes moderadas e não linearidade fraca, a frequência sem amortecimento sofre um desvio aproximado (Lindstedt–Poincaré):

\[ \omega_{\text{NL}} \approx \omega_n\!\left(1 + \frac{3\,\alpha A^2}{8k}\right). \]

Com amortecimento fraco (\(\zeta<1\)), aproximamos \(\omega_{d,\text{NL}} = \sqrt{\omega_{\text{NL}}^2 - (\tfrac{c}{2m})^2}\) e então \(T_{\text{NL}} \approx 2\pi/\omega_{d,\text{NL}}\).

Observações didáticas:

  • A expressão não linear é válida quando \(|\tfrac{3\alpha A^2}{8k}| \ll 1\) (correção pequena).
  • Para \(\zeta\ge 1\) não há oscilação: o período não é definido.
